# Astandy C# Core

Це високопродуктивний порт мережевого ядра бібліотеки **Astandy** на C# (.NET). Проєкт реалізує повну асинхронну архітектуру для взаємодії з сервером, забезпечуючи безпеку, стабільність та швидкість обміну даними, ідентичну оригінальному Python-ядру.

## 🚀 Основні можливості

* **Надійне мережеве ядро:** Асинхронне TCP-підключення з підтримкою SSL/TLS.
* **Безпека:** Реалізована криптографія AES-CBC з динамічним обміном ключами через RSA.
* **RPC-архітектура:** Універсальна система `SendRequestAsync` / `CallRpcAsync` для швидкого виклику віддалених процедур.
* **Система подій:** Гнучкий `EventDispatcher` для реєстрації типізованих обробників подій (SOLID).
* **Оптимізація:** Підтримка LZ4-декомпресії для швидкого опрацювання великих пакетів від сервера.

## 🏗 Архітектура
Проєкт побудований за модульним принципом:
* **`Services/`**: Набір сервісів (Hello, Handshake, Bolt, Player) для виконання бізнес-логіки.
* **`Events/`**: Диспетчер подій для обробки асинхронних оновлень.
* **`Utils/Parser.cs`**: Утиліти для серіалізації/десеріалізації Protobuf-повідомлень.
* **`AstandyCipher.cs`**: Модуль шифрування трафіку.
